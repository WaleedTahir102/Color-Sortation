# System Requirements

This section lists the hardware, software, and workspace requirements for the color sortation task.

## Hardware Requirements

The following hardware is required:

- Trossen Mobile AI Robot
- Onboard camera system
- Red colored blocks
- Blue colored blocks
- Green colored blocks
- Blocks of different sizes
- Control computer or laptop
- Stable power supply
- Working network connection

## Software Requirements

The following software tools are required:

- Trossen AI Data Collection UI
- Teleoperation interface
- Camera streaming and recording system
- Hugging Face account
- Robot training environment

## Workspace Requirements

The workspace should be prepared before recording episodes.

Make sure that:

- The robot has enough space to move safely.
- The colored block is visible to the camera.
- Lighting is clear and consistent.
- The background is not too confusing.
- The robot is placed on a stable surface.
- The camera view is checked before recording.
- Unnecessary objects are removed from the workspace.

## Safety Notes

Before starting data collection:

- Keep hands away from moving parts.
- Do not place objects too close to the robot body.
- Stop the robot if movement becomes unstable.
- Use a dry run before formal recording.
- Check camera and robot response before collecting data.
